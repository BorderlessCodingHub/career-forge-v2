# Career Forge Web — Next.js

App Router + TypeScript + Tailwind. Borderless tokens in `app/globals.css`.

## Dev

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000

## Env

Copy `.env.example` → `.env.local` and set `NEXT_PUBLIC_API_URL`.

## Deploy

Vercel project root: `apps/web` (see repo `vercel.json`).
